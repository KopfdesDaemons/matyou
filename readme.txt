== Description ==

This theme is inspired by the popular and modern Material Design.

== Installation ==

1. In your WordPress dashboard, navigate to "Appearance" > "Themes."
2. Click on "Add New" at the top of the page.
3. Click on "Upload Theme."
4. Choose the ZIP file of the MatYou theme that you downloaded from the official WordPress Themes website.
5. Click on "Install Now."
6. Activate the MatYou theme once the installation is complete.
